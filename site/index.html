Preston

A REST API framework for Node.js/Mongoose apps.

What is Preston?

Preston is a framework for building REST APIs from Mongoose models.

Why Preston?

Because the internet.

Extensibility

While Preston provides a very powerful API off-the-bat, you may want to add things such as authentication middleware, route-specific middlewares, or even custom queries. Preston allows you to shape your API the way you want it whilst providing a simple framework to do so.
