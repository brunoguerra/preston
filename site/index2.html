preston

REST API generator for Mongoose models


preston(mongoose.model('User'))
  .submodel(mongoose.model('Comment'));
preston.printRoutes();

Output


GET /users
POST /users
GET /users/:id
PUT /users/:id
PATCH /users/:id
DELETE /users/:id
GET /users/:id/comments
POST /users/:id/comments
GET /users/:id/comments/:sid
PUT /users/:id/comments/:sid
PATCH /users/:id/comments/:sid
DELETE /users/:id/comments/:sid
